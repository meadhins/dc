# Salah Dashboard

# About

This is a lightweight personal prayer-time dashboard built for daily use.

The goal is simple: provide quick awareness of the current prayer window and the time remaining without notifications, tracking, statistics, streaks, or other productivity features.

It displays:

* Current time, day, and date
* Prayer windows
* Remaining time in the current prayer
* Progress through the current prayer window
* Restricted prayer times

The dashboard is intentionally minimal and designed primarily for personal use.
